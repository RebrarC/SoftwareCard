## 🛠 Kullanılan Teknolojiler

- React Native
- Expo SDK 55
- JSX
- Component, Props, State
- FlatList
- Animated API
- StyleSheet (Flexbox)

---

## 🚀 Projeyi Çalıştır

**Bağımlılıkları yükle:**
```bash
npm install
```

**Uygulamayı başlat:**
```bash
npx expo start
```

**Web'de aç:**
```bash
npx expo start --web
```
